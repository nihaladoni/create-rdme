## ⚡ Getting started

1. Clone the repo

```sh
git clone https://github.com/nihaladoni/create-rdme.git
```

2. Change directory

```sh
 cd create-rdme
```

3. Install NPM packages

```sh
npm install
```

4. Run the app using command

```sh
sudo npm link
```
